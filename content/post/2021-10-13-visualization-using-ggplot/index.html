This dataset contains information of the value of commitments of each type of loan purpose from January 2018 to November 2018.

Here we try to show a total trend of the value of commitments.1

Figure 1: Total value of commitments trend

We can see a clear growth trend on the total financing commitment, so we can do a smoothing with this dataset. Before this, we need a transform of the current data table as the TIME feature here is not a variable that can us to fit the geom_smooth function.

Figure 2: Total value of commitments trend and Regression Plot

There is a significant growing trend in the plot above2, and with a 95% confidence band the gray area covers most of the points. However, we can see that some of the points are higher than the expected value, and there seems like a seasonal trend for these points. 3rd, 6th months are higher than other months, and there is also a growing trend after September. In this case we can guess that the total commitement may have seasonal behaviour, and we can test it if we have more data.

But each type of financing purpose actually make difference.

We can see from the pie chart below,3 Commercial Financing and Personal Financing took up most of the Financing commitment while the remaining two financing purpose even not taking up to 1%.
